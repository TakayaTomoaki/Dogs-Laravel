## タイトル・概要

【タイトル】

　Dogs

【URL】
   
　http://3.21.186.94/


【概要】

　Twitterのようなサイトになります。愛犬の兄弟を探すサイトを作成したいという思いから、犬主体のサイトにしています。

【目的】

　ポートフォリオ作成を通じて、開発をいちから体験することで、 自らの課題を見つけ、課題解決に向けた情報収集、実践、検証の試行錯誤を繰り返すことで技術習得を目指しています。


## 使用技術

- PHP 7.4.21
- Laravel 6.20.41 (LTS)
- Bootstrap 4.6.1
- MySQL 8.0.27
- apache 2.4.51
- AWS
  - EC2
